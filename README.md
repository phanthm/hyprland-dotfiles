## Installation

Install the following packages:

`pywal hyprpaper hypridle hyprlock waybar wlogout wofi nm-applet pavucontrol dunst nsxiv btop`

Make sure to also have the `Jetbrains Mono Font` installed.

## Setup

`mkdir ~/.config/btop \
~/.config/dunst \
~/.config/gtk-3.0 \
~/.config/gtk-4.0 \
~/.config/hypr \
~/.config/hypr/scripts \
~/.config/kitty \
~/.config/wal/templates \
~/.config/waybar \
~/.config/wlogout \
~/.config/wofi \
~/.local/share/icons/dunst`

`cd hyprland-dotfiles`

`cp -r config/* ~/.config/
cp -r local/* ~/.local/
cp .Xresources ~/`

Set the wallpaper through the wallpaper switcher once to generate colors.

Link the configs/colors generated by pywal:
`ln -sf ~/.cache/wal/dunstrc ~/.config/dunst/dunstrc
ln -sf ~/.cache/wal/gtk4.css ~/.config/gtk-4.0/colors.css
ln -sf ~/.cache/wal/colors-hyprland.conf ~/.config/hypr/colors.conf`
